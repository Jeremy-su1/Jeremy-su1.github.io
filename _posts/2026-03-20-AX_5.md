### LoRA
- Low-Rank 행렬을 사용하여 가중치 업데이트를 표현
- 𝛥W = Wa * Wb

```python
payload_base = {
    "use_lora_model": False, # not use lora
    "prompt": make_prompt(PROMPT),
    "height": HEIGHT,
    "width": WIDTH,

    "guidance_scale": 7.0,

    "num_images_per_prompt": 1,
```

LoRA 학습중에일어나는일
1. 학습 이미지(x₀)와 캡션을 가져옵니다.
2. 랜덤 timestep t를 고릅니다.
    -> 노이즈를 timestep에 따라 가중이 다르다. 많을수록 노이즈가 많다.
3. 해당 단계에 맞게 노이즈 ε를 샘플링해서 x_t를 만듭니다.
    -> 
4. 모델은 (x_t, t, text_embedding)을 입력받아 ε를 예측합니다.
    ->  예측값 ε
5. 예측한 ε과 실제 ε의 차이를 Loss로 계산하고, 이를 줄이는 방향으로 LoRA 가중치를 업데이트합니다.
    -> loss = | ε - 예측 ε |
